# Emerald-Logistics-Optimization
"A Data &amp; Business Analytics project optimizing Dublin-based freight logistics by reducing fuel waste and 'Ghost Bus' delay"


## 📌 Project Overview
This project addresses the "Ghost Bus" and logistics waste issue in the Irish market. 
Using **Python** and **SQL logic**, I analyzed RTPI (Real-Time Passenger Information) 
logs to identify cancelled services and calculate delay impacts during adverse weather.

## 🛠️ Tech Stack
- **Python:** Pandas for data cleaning and feature engineering.
- **Data:** Dublin Open Data (RTPI) & Irish Weather Records.
- **Logic:** Decision trees for cost-benefit analysis of hybrid-work vs. salary raises.

## 📊 Business Insights
- Identified a **15% increase** in "Ghost Buses" during heavy rain windows (08:00-09:00).
- Created a `Status` classification system to automate service-level reporting.
